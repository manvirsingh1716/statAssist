# ModelPath Execution Roadmap

## Phase 1 (0-4 weeks)

- Finalize guided workflow UX shell
- Stabilize auth (email + Google)
- Add project/workspace persistence model
- Add health checks, logging, and environment conventions

## Phase 2 (5-8 weeks)

- Add track templates (regression, classification)
- Add instructor-facing assignment model
- Add run history and report export

## Phase 3 (9-12 weeks)

- Add asynchronous training jobs and queue workers
- Add learner analytics and cohort progress insights
- Add versioned model registry and artifact storage

## KPIs

- First model completion rate
- Day-7 return rate
- Lesson completion rate
- Instructor activation rate
